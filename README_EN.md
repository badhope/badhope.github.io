## 🚀 Getting Started

### Prerequisites

- Node.js >= 18.0.0
- npm >= 9.0.0 or yarn >= 1.22.0

### Installation

```bash
# Clone the repository
git clone https://github.com/badhope/badhope.github.io.git
cd badhope.github.io

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Start production server
npm run start
```

### Environment Variables

Copy `.env.example` to `.env.local` and configure:

```env
# AI Model API Keys (optional)
NEXT_PUBLIC_OPENAI_API_KEY=your-openai-key
NEXT_PUBLIC_DEEPSEEK_API_KEY=your-deepseek-key
NEXT_PUBLIC_ZHIPU_API_KEY=your-zhipu-key
```

## ⚙️ Configuration

### AI Assistant

Edit `src/config/ai.ts` to configure AI models:

```typescript
const aiModels = [
  {
    name: 'OpenAI',
    apiKey: 'your-api-key-here',
    enabled: true,
  },
  // More models: DeepSeek, Zhipu AI, Tongyi, Wenxin...
];
```

### Social Links

Edit `src/config/social.ts` to update social links:

```typescript
const socialLinks = [
  { name: 'GitHub', url: 'https://github.com/your-username' },
  { name: 'Twitter', url: 'https://twitter.com/your-handle' },
  // ...
];
```

### Tool Collection

Edit `src/config/tools.ts` to add/remove tools.
