## 🚀 快速开始

### 环境要求

- Node.js >= 18.0.0
- npm >= 9.0.0 或 yarn >= 1.22.0

### 安装步骤

```bash
# 克隆仓库
git clone https://github.com/badhope/badhope.github.io.git
cd badhope.github.io

# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 启动生产服务器
npm run start
```

### 环境变量配置

复制 `.env.example` 为 `.env.local` 并配置：

```env
# AI 模型 API Keys（可选）
NEXT_PUBLIC_OPENAI_API_KEY=your-openai-key
NEXT_PUBLIC_DEEPSEEK_API_KEY=your-deepseek-key
NEXT_PUBLIC_ZHIPU_API_KEY=your-zhipu-key
```

## ⚙️ 配置指南

### AI 助手配置

编辑 `src/config/ai.ts` 配置 AI 模型：

```typescript
const aiModels = [
  {
    name: 'OpenAI',
    apiKey: 'your-api-key-here',
    enabled: true,
  },
  // 支持更多模型：DeepSeek, 智谱AI, 通义千问, 文心一言...
];
```

### 社交链接配置

编辑 `src/config/social.ts` 更新社交链接：

```typescript
const socialLinks = [
  { name: 'GitHub', url: 'https://github.com/your-username' },
  { name: 'Twitter', url: 'https://twitter.com/your-handle' },
  // ...
];
```

### 工具集合配置

编辑 `src/config/tools.ts` 添加/移除工具。
